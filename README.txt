# Game of Life

"Game of Life" - это реализация известного клеточного автомата, созданного Джоном Конвеем. Игра позволяет пользователям задавать начальную конфигурацию и наблюдать за эволюцией клеток на сетке.

## Требования

- Компилятор g++
- SDL2
- SDL2_ttf

### Установка

1. В директории проекта в терминале пропишите следующую команду:
    ```sh
    g++ -std=c++20 -lSDL2 -lSDL2_ttf main.cpp -o Game_Of_Life
    ```


### Запуск проекта

1. Запустите приложение:
    ```sh
    ./Game_Of_Life
    ```
Описание

	Сразу после запуска открывается меню с настройками для игры. Можно настроить следующие параметры:
		-Ширину
		-Высоту
		-Скорость
	Так же в меню есть информация об управлении во время игры:
		-При нажатии на кнопку "левой кнопки мыши(ЛКМ)" появляется жизнь
		-При нажатии на кнопку "правой кнопки мыши(ПКМ)" жизнь стирается
		-При нажатии на "пробел" игра запускается, повторное нажатие останавливает игру
		-При нажатии на кнопку "r"(reset) игра перезапускается
